## 📄 개요

- **국회사무처 의안정보 공공데이터를 이용한 국회의안, 국회의원 정보제공 및 의안토론 서비스.**

    국회에 대한 공공연한 사회적 불신을 해소하기 위해 공공데이터를 이용하여 국회의원들이 발의하는 법안을 보여주고 간단한 찬성 반대 투표와 댓글을 통한 찬반 토론이 가능한 시스템을 개발. 추가로 현재 국회의원들을 검색하여 간단한 정보들을 제공하는 서비스이다.
